# Yelp Customer Satisfaction Analysis

This project investigates factors associated with restaurant customer satisfaction using Yelp review data.

## Tools
- PostgreSQL
- SQL
- R
- Logistic Regression
- Decision Tree
- Random Forest

## Workflow
1. Extract data using SQL
2. Export dataset from PostgreSQL
3. Analyze data in R
4. Build predictive models
5. Generate managerial insights
